<p>Wikipedia is a free <a href="/wiki/Encyclopedia" title="Encyclopedia">encyclopedia</a>, written <a href="/wiki/Collaborative_editing" title="Collaborative editing">collaboratively</a> by <a href="/wiki/Wikipedia:Who_writes_Wikipedia" class="mw-redirect" title="Wikipedia:Who writes Wikipedia">the people who use it</a>. It is a special type of website designed to make collaboration easy, called a <a href="/wiki/Wiki" title="Wiki">wiki</a>. Many people are constantly improving Wikipedia, making thousands of changes per hour. All of these changes are recorded in <a href="/wiki/Help:Page_history" title="Help:Page history">article histories</a> and <a href="/wiki/Help:Recent_changes" title="Help:Recent changes">recent changes</a>. For a more detailed account of the project, see <b><a href="/wiki/Wikipedia:About" title="Wikipedia:About">About Wikipedia</a></b>.<br />

<p><b>Don't be afraid to edit</b> – <i>anyone</i> can edit almost every page, and we are encouraged to <b><a href="/wiki/Wikipedia:Be_bold" title="Wikipedia:Be bold">be bold</a></b>! Find something that can be improved and make it better—for example, spelling, grammar, rewriting for readability, adding content, or removing non-constructive edits. If you wish to add new facts, please try to provide <a href="/wiki/Help:Introduction_to_referencing/1" class="mw-redirect" title="Help:Introduction to referencing/1">references</a> so they may be <a href="/wiki/Wikipedia:Verifiability" title="Wikipedia:Verifiability">verified</a>, or suggest them on the article's <a href="/wiki/Help:Introduction_to_talk_pages" class="mw-redirect" title="Help:Introduction to talk pages">discussion page</a>. Changes to controversial topics and Wikipedia's main pages should usually be discussed first. <b><a href="/wiki/Wikipedia:Contributing_to_Wikipedia" title="Wikipedia:Contributing to Wikipedia">Contributing to Wikipedia</a></b> will provide you with resources on all the basics needed to use, comment on, and contribute to Wikipedia.</p>
<p>Remember – you can't break Wikipedia; all edits can be reversed, fixed or improved later. Wikipedia is <a href="/wiki/Wikipedia:Editing_policy" title="Wikipedia:Editing policy">allowed to be imperfect</a>. So go ahead, edit an article and help make Wikipedia the best information source on the Internet!</p>
<p><b>Donate</b> – Wikipedia is free to use, but relies on donations and grants to do so. Please consider donating using the <b>Donate to Wikipedia</b> link on the left to help with the project's running costs and expansion.<br />
<br /></p>
