<h2><span class="mw-headline" id="The_pioneers">The pioneers</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=List_of_pioneers_in_computer_science&amp;action=edit&amp;section=1" title="Edit section: The pioneers">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<table class="wikitable sortable">
<tr>
<th><abbr title="Achievement">Ach.</abbr> Date</th>
<th>Person</th>
<th class="unsortable">Achievement</th>
</tr>
<tr>
<td style="width:90px">0830~</td>
<td style="width:120px"><a href="/wiki/Mu%E1%B8%A5ammad_ibn_M%C5%ABs%C4%81_al-Khw%C4%81rizm%C4%AB" class="mw-redirect" title="Muḥammad ibn Mūsā al-Khwārizmī">Al-Khwārizmī</a><sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup><sup id="cite_ref-2" class="reference"><a href="#cite_note-2">[2]</a></sup><sup id="cite_ref-3" class="reference"><a href="#cite_note-3">[3]</a></sup></td>
<td>The term "algorithm" is derived from the <a href="/wiki/Algorism" title="Algorism">algorism</a>, the technique of performing arithmetic with <a href="/wiki/Arabic" title="Arabic">Arabic</a> numerals developed by al-Khwarizmi.</td>
</tr>
<tr>
<td>1944</td>
<td><a href="/wiki/Howard_Aiken" class="mw-redirect" title="Howard Aiken">Howard Aiken</a></td>
<td>Conceived and codesigned the <a href="/wiki/Harvard_Mark_I" title="Harvard Mark I">Harvard Mark I</a>.</td>
</tr>
<tr>
<td>1970</td>
<td><a href="/wiki/Frances_E._Allen" title="Frances E. Allen">Frances E. Allen</a></td>
<td>Developed <a href="/wiki/Bit_vector" class="mw-redirect" title="Bit vector">bit vector</a> notation and program <a href="/wiki/Control_flow_graph" title="Control flow graph">control flow graphs</a>.</td>
</tr>
<tr>
<td>1939</td>
<td><a href="/wiki/John_Vincent_Atanasoff" title="John Vincent Atanasoff">John Atanasoff</a></td>
<td>Built the first electronic digital computer, the <a href="/wiki/Atanasoff%E2%80%93Berry_Computer" class="mw-redirect" title="Atanasoff–Berry Computer">Atanasoff–Berry Computer</a>, though it was neither programmable nor <a href="/wiki/Turing-completeness" class="mw-redirect" title="Turing-completeness">Turing-complete</a>.</td>
</tr>
<tr>
<td>1822<br />
1837</td>
<td><a href="/wiki/Charles_Babbage" title="Charles Babbage">Charles Babbage</a></td>
<td>Originated the concept of a programmable general-purpose computer. Designed the <a href="/wiki/Analytical_Engine" title="Analytical Engine">Analytical Engine</a> and built a prototype for a less powerful <a href="/wiki/Difference_engine" title="Difference engine">mechanical calculator</a>.</td>
</tr>
<tr>
<td>1954<br />
1963</td>
<td><a href="/wiki/John_Backus" title="John Backus">John Backus</a></td>
<td>Led the team that created <a href="/wiki/FORTRAN" class="mw-redirect" title="FORTRAN">FORTRAN</a> (<i>For</i>mula <i>Tran</i>slation), the first practical high-level programming language, and he formulated the <a href="/wiki/Backus%E2%80%93Naur_form" title="Backus–Naur form">Backus–Naur form</a> that described the formal language <a href="/wiki/Syntax" title="Syntax">syntax</a>.</td>
</tr>
<tr>
<td>1946</td>
<td><a href="/wiki/Jean_Bartik" title="Jean Bartik">Jean Bartik</a></td>
<td>One of the first <a href="/wiki/Computer_programming#History" title="Computer programming">computer programmers</a>, on <a href="/wiki/ENIAC" title="ENIAC">ENIAC</a> (1946), an early <a href="/wiki/Vacuum_tube" title="Vacuum tube">vacuum tube</a> <a href="/wiki/Vacuum_tube#Use_in_electronic_computers" title="Vacuum tube">computer</a> back when "programming" involved using cables, dials, and switches to physically rewire the machine. Worked with <a href="/wiki/John_Mauchly" title="John Mauchly">John Mauchly</a> toward <a href="/wiki/BINAC" title="BINAC">BINAC</a> (1949), <a href="/wiki/EDVAC" title="EDVAC">EDVAC</a> (1949), <a href="/wiki/UNIVAC" title="UNIVAC">UNIVAC</a> (1951) to develop early "<a href="/wiki/Stored_program" class="mw-redirect" title="Stored program">Stored program</a>" computers.</td>
</tr>
<tr>
<td>1989<br />
1990</td>
<td><a href="/wiki/Tim_Berners-Lee" title="Tim Berners-Lee">Tim Berners-Lee</a></td>
<td>Invented <a href="/wiki/Worldwide_web" class="mw-redirect" title="Worldwide web">worldwide web</a>. With <a href="/wiki/Robert_Cailliau" title="Robert Cailliau">Robert Cailliau</a>, sent first HTTP communication between client and server.</td>
</tr>
<tr>
<td>1966</td>
<td><a href="/wiki/Corrado_B%C3%B6hm" title="Corrado Böhm">Corrado Böhm</a></td>
<td>Theorized of the concept of <a href="/wiki/Structured_programming" title="Structured programming">Structured programming</a>.</td>
</tr>
<tr>
<td>1847<br />
1854</td>
<td><a href="/wiki/George_Boole" title="George Boole">George Boole</a></td>
<td>Formalized <a href="/wiki/Boolean_algebra_(logic)" class="mw-redirect" title="Boolean algebra (logic)">Boolean algebra</a>, the basis for <a href="/wiki/Digital_logic" class="mw-redirect" title="Digital logic">digital logic</a> and computer science.</td>
</tr>
<tr>
<td>1947</td>
<td><a href="/wiki/Kathleen_Booth" title="Kathleen Booth">Kathleen Booth</a></td>
<td>Invented the first <a href="/wiki/Assembly_language" title="Assembly language">assembly language</a>.</td>
</tr>
<tr>
<td>1969<br />
1978</td>
<td><a href="/wiki/Per_Brinch_Hansen" title="Per Brinch Hansen">Per Brinch Hansen</a></td>
<td>Developed the <a href="/wiki/RC_4000_Multiprogramming_System" class="mw-redirect" title="RC 4000 Multiprogramming System">RC 4000 multiprogramming system</a> which introduced the concept of an <a href="/wiki/Kernel_(operating_system)" title="Kernel (operating system)">operating system kernel</a> and the separation of policy and mechanism, effectively the first <a href="/wiki/Microkernel" title="Microkernel">microkernel</a> architecture.<sup id="cite_ref-:0_4-0" class="reference"><a href="#cite_note-:0-4">[4]</a></sup> Co-developed the <a href="/wiki/Monitor_(synchronization)" title="Monitor (synchronization)">monitor</a> with <a href="/wiki/Tony_Hoare" title="Tony Hoare">Tony Hoare</a>, and created the first monitor implementation.<sup id="cite_ref-5" class="reference"><a href="#cite_note-5">[5]</a></sup> Implemented the first form of <a href="/wiki/Remote_procedure_call" title="Remote procedure call">remote procedure call</a> in the RC 4000,<sup id="cite_ref-:0_4-1" class="reference"><a href="#cite_note-:0-4">[4]</a></sup> and was first to propose remote procedure calls as a structuring concept for <a href="/wiki/Distributed_computing" title="Distributed computing">distributed computing</a>.<sup id="cite_ref-6" class="reference"><a href="#cite_note-6">[6]</a></sup></td>
</tr>
<tr>
<td>1959<br />
1995</td>
<td><a href="/wiki/Fred_Brooks" title="Fred Brooks">Fred Brooks</a></td>
<td>Manager of IBM <a href="/wiki/System/360" class="mw-redirect" title="System/360">System/360</a> and <a href="/wiki/OS/360" class="mw-redirect" title="OS/360">OS/360</a> projects; author of <a href="/wiki/The_Mythical_Man-Month" title="The Mythical Man-Month">The Mythical Man-Month</a>.</td>
</tr>
<tr>
<td>1958</td>
<td><a href="/wiki/Nikolay_Brusentsov" title="Nikolay Brusentsov">Nikolay Brusentsov</a></td>
<td>Built <a href="/wiki/Ternary_computer" title="Ternary computer">ternary computer</a> <a href="/wiki/Setun" title="Setun">Setun</a>.</td>
</tr>
<tr>
<td>1930</td>
<td><a href="/wiki/Vannevar_Bush" title="Vannevar Bush">Vannevar Bush</a></td>
<td>Analogue computing pioneer. Originator of the <a href="/wiki/Memex" title="Memex">Memex</a> concept, which led to the development of <a href="/wiki/Hypertext" title="Hypertext">Hypertext</a>.</td>
</tr>
<tr>
<td>1951</td>
<td><a href="/wiki/David_Caminer" title="David Caminer">David Caminer</a></td>
<td>With <a href="/wiki/John_Pinkerton_(computer_designer)" title="John Pinkerton (computer designer)">John Pinkerton</a>, developed the <a href="/wiki/LEO_computer" class="mw-redirect" title="LEO computer">LEO computer</a> the first business computer, for <a href="/wiki/J._Lyons_and_Co" class="mw-redirect" title="J. Lyons and Co">J. Lyons and Co</a></td>
</tr>
<tr>
<td>1978</td>
<td><a href="/wiki/Vint_Cerf" title="Vint Cerf">Vint Cerf</a></td>
<td>With <a href="/wiki/Bob_Kahn" title="Bob Kahn">Bob Kahn</a>, designed the <a href="/wiki/Transmission_Control_Protocol" title="Transmission Control Protocol">Transmission Control Protocol</a> and <a href="/wiki/Internet_Protocol" title="Internet Protocol">Internet Protocol</a> (<a href="/wiki/TCP/IP" class="mw-redirect" title="TCP/IP">TCP/IP</a>), the primary data <a href="/wiki/Communication_protocols" class="mw-redirect" title="Communication protocols">communication protocols</a> of the <a href="/wiki/Internet" title="Internet">Internet</a> and other <a href="/wiki/Computer_networks" class="mw-redirect" title="Computer networks">computer networks</a>.</td>
</tr>
<tr>
<td>1956</td>
<td><a href="/wiki/Noam_Chomsky" title="Noam Chomsky">Noam Chomsky</a></td>
<td>Made contributions to computer science with his work in <a href="/wiki/Linguistics" title="Linguistics">linguistics</a>. He developed <a href="/wiki/Chomsky_hierarchy" title="Chomsky hierarchy">Chomsky hierarchy</a>, a discovery which has directly impacted <a href="/wiki/Programming_language_theory" title="Programming language theory">programming language theory</a> and other branches of computer science.</td>
</tr>
<tr>
<td>1936</td>
<td><a href="/wiki/Alonzo_Church" title="Alonzo Church">Alonzo Church</a></td>
<td>Founded contributions to theoretical computer science, specifically for the development of the <a href="/wiki/Lambda_calculus" title="Lambda calculus">lambda calculus</a> and the discovery of the <a href="/wiki/Undecidable_problem" title="Undecidable problem">undecidability problem</a> within it.</td>
</tr>
<tr>
<td>1962</td>
<td><a href="/wiki/Wesley_A._Clark" title="Wesley A. Clark">Wesley A. Clark</a></td>
<td>Designed <a href="/wiki/LINC" title="LINC">LINC</a>, the first functional computer scaled down and priced for the individual user. Put in service in 1963, many of its features are seen as prototypes of what were to be essential elements of personal computers.</td>
</tr>
<tr>
<td>1981</td>
<td><a href="/wiki/Edmund_M._Clarke" title="Edmund M. Clarke">Edmund M. Clarke</a></td>
<td>Developed <a href="/wiki/Model_checking" title="Model checking">model checking</a> and formal verification of software and hardware together with <a href="/wiki/E._Allen_Emerson" title="E. Allen Emerson">E. Allen Emerson</a>.</td>
</tr>
<tr>
<td>1970</td>
<td><a href="/wiki/Edgar_F._Codd" title="Edgar F. Codd">Edgar F. Codd</a></td>
<td>Proposed and formalized the <a href="/wiki/Relational_model" title="Relational model">relational model</a> of data management, the theoretical basis of <a href="/wiki/Relational_databases" class="mw-redirect" title="Relational databases">relational databases</a>.</td>
</tr>
<tr>
<td>1971</td>
<td><a href="/wiki/Stephen_Cook" title="Stephen Cook">Stephen Cook</a></td>
<td>Formalized the notion of NP-completeness, inspiring a great deal of research in <a href="/wiki/Computational_complexity_theory" title="Computational complexity theory">computational complexity theory</a>.</td>
</tr>
<tr>
<td>1965</td>
<td><a href="/wiki/James_Cooley" title="James Cooley">James Cooley</a></td>
<td>With <a href="/wiki/John_W._Tukey" class="mw-redirect" title="John W. Tukey">John W. Tukey</a>, created the <a href="/wiki/Cooley-Tukey_FFT_algorithm" class="mw-redirect" title="Cooley-Tukey FFT algorithm">fast Fourier transform</a>.</td>
</tr>
<tr>
<td>1962</td>
<td><a href="/wiki/Ole-Johan_Dahl" title="Ole-Johan Dahl">Ole-Johan Dahl</a></td>
<td>With <a href="/wiki/Kristen_Nygaard" title="Kristen Nygaard">Kristen Nygaard</a>, invented the proto-object oriented language <a href="/wiki/SIMULA" class="mw-redirect" title="SIMULA">SIMULA</a>.</td>
</tr>
<tr>
<td>1968</td>
<td><a href="/wiki/Edsger_Dijkstra" class="mw-redirect" title="Edsger Dijkstra">Edsger Dijkstra</a></td>
<td>Made advances in algorithms, pioneered and coined the term <a href="/wiki/Structured_programming" title="Structured programming">structured programming</a>, invented the <a href="/wiki/Semaphore_(programming)" title="Semaphore (programming)">semaphore</a>, and famously suggested that the <a href="/wiki/Goto_(command)" class="mw-redirect" title="Goto (command)">GOTO statement</a> should be <a href="/wiki/Considered_harmful" title="Considered harmful">considered harmful</a>.</td>
</tr>
<tr>
<td>1943<br />
1951</td>
<td><a href="/wiki/J._Presper_Eckert" title="J. Presper Eckert">J. Presper Eckert</a></td>
<td>With <a href="/wiki/John_Mauchly" title="John Mauchly">John Mauchly</a>, designed and built the <a href="/wiki/ENIAC" title="ENIAC">ENIAC</a>, the first modern (all electronic, Turing-complete) computer, and the <a href="/wiki/UNIVAC_I" title="UNIVAC I">UNIVAC I</a>, the first commercially available computer.</td>
</tr>
<tr>
<td>1981</td>
<td><a href="/wiki/E._Allen_Emerson" title="E. Allen Emerson">E. Allen Emerson</a></td>
<td>Developed <a href="/wiki/Model_checking" title="Model checking">model checking</a> and formal verification of software and hardware together with <a href="/wiki/Edmund_M._Clarke" title="Edmund M. Clarke">Edmund M. Clarke</a>.</td>
</tr>
<tr>

<tr>
<td>1985</td>
<td><a href="/wiki/Radia_Perlman" title="Radia Perlman">Radia Perlman</a></td>
<td>Invented the <a href="/wiki/Spanning-tree_protocol" class="mw-redirect" title="Spanning-tree protocol">spanning-tree protocol</a> (STP), which is fundamental to the operation of <a href="/wiki/Network_bridge" class="mw-redirect" title="Network bridge">network bridges</a>, while working for <a href="/wiki/Digital_Equipment_Corporation" title="Digital Equipment Corporation">Digital Equipment Corporation</a></td>
</tr>
<tr>
<td>1964</td>
<td><a href="/wiki/Pier_Giorgio_Perotto" title="Pier Giorgio Perotto">Pier Giorgio Perotto</a></td>
<td>Designer of <a href="/wiki/Olivetti_Programma_101" class="mw-redirect" title="Olivetti Programma 101">Olivetti Programma 101</a>, the first <a href="/wiki/Personal_computer" title="Personal computer">personal computer</a>.</td>
</tr>
<tr>
<td>1932</td>
<td><a href="/wiki/R%C3%B3zsa_P%C3%A9ter" title="Rózsa Péter">Rózsa Péter</a></td>
<td>Published a series of papers grounding <a href="/wiki/Recursion_theory" class="mw-redirect" title="Recursion theory">recursion theory</a> as a separate area of mathematical research, setting the foundation for theoretical computer science.</td>
</tr>
<tr>
<td>1995</td>
<td><a href="/wiki/Rosalind_Picard" title="Rosalind Picard">Rosalind Picard</a></td>
<td>Founded <a href="/wiki/Affective_Computing" class="mw-redirect" title="Affective Computing">Affective Computing</a>, and laid the foundations for giving computers skills of emotional intelligence.</td>
</tr>
<tr>
<td>1936</td>
<td><a href="/wiki/Emil_Leon_Post" title="Emil Leon Post">Emil L. Post</a></td>
<td>Developed the <a href="/wiki/Post%E2%80%93Turing_machine" title="Post–Turing machine">Post machine</a> as a model of computation, independently of Turing. Known also for developing <a href="/wiki/Truth_tables" class="mw-redirect" title="Truth tables">truth tables</a>, the <a href="/wiki/Post_correspondence_problem" title="Post correspondence problem">Post correspondence problem</a> used in recursion theory as well as proving what is known as <a href="/wiki/Post%27s_theorem" title="Post's theorem">Post's theorem</a>.</td>
</tr>
<tr>
<td>1967</td>
<td><a href="/wiki/Dennis_Ritchie" title="Dennis Ritchie">Dennis Ritchie</a></td>
<td>With <a href="/wiki/Ken_Thompson" title="Ken Thompson">Ken Thompson</a>, pioneered the <a href="/wiki/C_(programming_language)" title="C (programming language)">C programming language</a> and the <a href="/wiki/Unix" title="Unix">Unix</a> computer operating system at Bell Labs.</td>
</tr>
<tr>
<td>1967</td>
<td><a href="/wiki/Ken_Thompson" title="Ken Thompson">Ken Thompson</a></td>
<td>Created the <a href="/wiki/Unix" title="Unix">Unix</a> operating system, the <a href="/wiki/B_programming_language" class="mw-redirect" title="B programming language">B programming language</a>, <a href="/wiki/Plan_9" class="mw-disambig" title="Plan 9">Plan 9</a> operating system, the first computer chess game and the <a href="/wiki/UTF-8" title="UTF-8">UTF-8</a> encoding at Bell Labs and the <a href="/wiki/Go_programming_language" class="mw-redirect" title="Go programming language">Go programming language</a> at Google.</td>
</tr>
<tr>
<td>1958–1960</td>
<td><a href="/wiki/Saul_Rosen" title="Saul Rosen">Saul Rosen</a></td>
<td>Designed the software of the first transistor-based computer. Also influenced the ALGOL programming language.</td>
</tr>
<tr>
<td>1910</td>
<td><a href="/wiki/Bertrand_Russell" title="Bertrand Russell">Bertrand Russell</a></td>
<td>Made contributions to computer science with his work on <a href="/wiki/Mathematical_logic" title="Mathematical logic">mathematical logic</a> (example: <a href="/wiki/Truth_function" title="Truth function">truth function</a>). Introduced the notion of <a href="/wiki/Type_theory" title="Type theory">type theory</a>. He also introduced <a href="/wiki/Type_system" title="Type system">type system</a> (along with <a href="/wiki/Alfred_North_Whitehead" title="Alfred North Whitehead">Alfred North Whitehead</a>) in his work, <a href="/wiki/Principia_Mathematica" title="Principia Mathematica">Principia Mathematica</a>.</td>
</tr>
<tr>
<td>1975</td>
<td><a href="/wiki/Gerard_Salton" title="Gerard Salton">Gerard Salton</a></td>
<td>A pioneer of automatic <a href="/wiki/Information_retrieval" title="Information retrieval">information retrieval</a>, who proposed the <a href="/wiki/Vector_space_model" title="Vector space model">vector space model</a> and the <a href="/wiki/Inverted_index" title="Inverted index">inverted index</a>.</td>
</tr>
<tr>
<td>1963<br />
1973</td>
<td><a href="/wiki/Tadashi_Sasaki_(engineer)" title="Tadashi Sasaki (engineer)">Tadashi Sasaki</a></td>
<td><a href="/wiki/Sharp_Corporation" title="Sharp Corporation">Sharp</a> engineer who conceived a single-chip <a href="/wiki/Microprocessor" title="Microprocessor">microprocessor</a> <a href="/wiki/CPU" class="mw-redirect" title="CPU">CPU</a>, presenting the idea to <a href="/wiki/Busicom" title="Busicom">Busicom</a> and <a href="/wiki/Intel" title="Intel">Intel</a> in 1968, influencing the first commercial microprocessor, the <a href="/wiki/Intel_4004" title="Intel 4004">Intel 4004</a>. He also developed <a href="/wiki/LCD" class="mw-redirect" title="LCD">LCD</a> <a href="/wiki/Calculator" title="Calculator">calculators</a> at Sharp.<sup id="cite_ref-sasaki_22-0" class="reference"><a href="#cite_note-sasaki-22">[22]</a></sup></td>
</tr>
<tr>
<td>1937<br />
1948</td>
<td><a href="/wiki/Claude_E._Shannon" class="mw-redirect" title="Claude E. Shannon">Claude Shannon</a></td>
<td>Founded <a href="/wiki/Information_theory" title="Information theory">information theory</a>, and laid foundations for practical digital circuit design.</td>
</tr>
<tr>
<td>1968<br />
1980</td>
<td><a href="/wiki/Masatoshi_Shima" title="Masatoshi Shima">Masatoshi Shima</a></td>
<td>Designed the <a href="/wiki/Intel_4004" title="Intel 4004">Intel 4004</a>, the first commercial <a href="/wiki/Microprocessor" title="Microprocessor">microprocessor</a>,<sup id="cite_ref-tout1_23-0" class="reference"><a href="#cite_note-tout1-23">[23]</a></sup><sup id="cite_ref-ieee_24-0" class="reference"><a href="#cite_note-ieee-24">[24]</a></sup> as well as the <a href="/wiki/Intel_8080" title="Intel 8080">Intel 8080</a>, <a href="/wiki/Zilog_Z80" title="Zilog Z80">Zilog Z80</a> and <a href="/wiki/Zilog_Z8000" title="Zilog Z8000">Zilog Z8000</a> microprocessors, and the <a href="/wiki/Intel_8259" title="Intel 8259">Intel 8259</a>, <a href="/wiki/8255" class="mw-redirect" title="8255">8255</a>, <a href="/wiki/8253" class="mw-redirect" title="8253">8253</a>, <a href="/wiki/Intel_8257" title="Intel 8257">8257</a> and <a href="/wiki/8251" class="mw-redirect" title="8251">8251</a> chips.<sup id="cite_ref-25" class="reference"><a href="#cite_note-25">[25]</a></sup></td>
</tr>
<tr>
<td>1981<br />
1991</td>
<td><a href="/wiki/Shiraz_Shivji" title="Shiraz Shivji">Shiraz Shivji</a></td>
<td>Designed the <a href="/wiki/Atari_ST" title="Atari ST">Atari ST</a><sup id="cite_ref-26" class="reference"><a href="#cite_note-26">[26]</a></sup> and <a href="/wiki/Commodore_64" title="Commodore 64">Commodore 64</a> computers,<sup id="cite_ref-classic_27-0" class="reference"><a href="#cite_note-classic-27">[27]</a></sup> as well as an early <a href="/wiki/Touchscreen" title="Touchscreen">touchscreen</a> <a href="/wiki/Tablet_computer" title="Tablet computer">tablet computer</a>, the Momenta Pentop Computer introduced in 1991.<sup id="cite_ref-28" class="reference"><a href="#cite_note-28">[28]</a></sup></td>
</tr>
<tr>
<td>1964</td>
<td><a href="/wiki/Glenda_Schroeder" title="Glenda Schroeder">Glenda Schroeder</a></td>
<td>Implemented the first <a href="/wiki/Command_line" class="mw-redirect" title="Command line">command line</a> <a href="/wiki/User_interface" title="User interface">user interface</a> <a href="/wiki/Shell_(computing)" title="Shell (computing)">shell</a>.<sup id="cite_ref-29" class="reference"><a href="#cite_note-29">[29]</a></sup> She also described the first <a href="/wiki/E-mail" class="mw-redirect" title="E-mail">e-mail</a> implementation <sup id="cite_ref-30" class="reference"><a href="#cite_note-30">[30]</a></sup> in 1964-65 (with <a href="/w/index.php?title=Pat_Crisman&amp;action=edit&amp;redlink=1" class="new" title="Pat Crisman (page does not exist)">Pat Crisman</a> and <a href="/wiki/Louis_Pouzin" title="Louis Pouzin">Louis Pouzin</a>) by outlining a system of notifying users about backups of files.<sup id="cite_ref-31" class="reference"><a href="#cite_note-31">[31]</a></sup></td>
</tr>
<tr>
<td>1956<br />
1957</td>
<td><a href="/wiki/Herbert_A._Simon" title="Herbert A. Simon">Herbert A. Simon</a></td>
<td>A political scientist and economist who pioneered <a href="/wiki/Artificial_intelligence" title="Artificial intelligence">artificial intelligence</a>. Co-creator of the <a href="/wiki/Logic_Theory_Machine" class="mw-redirect" title="Logic Theory Machine">Logic Theory Machine</a> and the <a href="/wiki/General_Problem_Solver" title="General Problem Solver">General Problem Solver</a> programs.</td>
</tr>
<tr>
<td>1970<br />
1990</td>
<td><a href="/wiki/Richard_Stallman" title="Richard Stallman">Richard Stallman</a></td>
<td>Stallman launched the <a href="/wiki/GNU_Project" title="GNU Project">GNU Project</a> in September 1983 to create a Unix-like computer operating system composed entirely of free software. With this, he also launched the free software movement.</td>
</tr>
<tr>
<td>1982</td>
<td><a href="/wiki/Michael_Stonebraker" title="Michael Stonebraker">Michael Stonebraker</a></td>
<td>Researcher at MIT’s Computer Science and Artificial Intelligence Laboratory (CSAIL) who revolutionized the field of database management systems (DBMSs) and founded multiple successful database companies</td>
</tr>
<tr>
<td>1963</td>
<td><a href="/wiki/Ivan_Sutherland" title="Ivan Sutherland">Ivan Sutherland</a></td>
<td>Author of <a href="/wiki/Sketchpad" title="Sketchpad">Sketchpad</a>, the ancestor of modern computer-aided drafting (CAD) programs and one of the early examples of <a href="/wiki/Object-oriented_programming" title="Object-oriented programming">object-oriented programming</a>.</td>
</tr>
<tr>
<td>1978<br />
1982</td>
<td><a href="https://sv.wikipedia.org/wiki/Yasaharu_Terakura" class="extiw" title="sv:Yasaharu Terakura">Yasaharu "Yash" Terakura</a></td>
<td><a href="/wiki/Commodore_International" title="Commodore International">Commodore Japan</a> engineer who designed the <a href="/wiki/Commodore_PET" title="Commodore PET">color PET</a>, <a href="/wiki/VIC-20" class="mw-redirect" title="VIC-20">VIC-20</a>, <a href="/wiki/MAX_Machine" class="mw-redirect" title="MAX Machine">MAX Machine</a> and <a href="/wiki/Commodore_64" title="Commodore 64">C64</a> computers.<sup id="cite_ref-32" class="reference"><a href="#cite_note-32">[32]</a></sup></td>
</tr>
<tr>
<td>1965</td>
<td><a href="/wiki/John_W._Tukey" class="mw-redirect" title="John W. Tukey">John W. Tukey</a></td>
<td>With <a href="/wiki/James_Cooley" title="James Cooley">James Cooley</a>, created the <a href="/wiki/Cooley-Tukey_FFT_algorithm" class="mw-redirect" title="Cooley-Tukey FFT algorithm">fast Fourier transform</a>.</td>
</tr>
<tr>
<td>1936</td>
<td><a href="/wiki/Alan_Turing" title="Alan Turing">Alan Turing</a></td>
<td>Made several founding contributions to computer science, including the <a href="/wiki/Turing_machine" title="Turing machine">Turing machine</a> computational model, the conceiving of the stored program concept and the designing of the high-speed <a href="/wiki/Automatic_Computing_Engine" title="Automatic Computing Engine">ACE</a> design. Widely considered as the father of <a href="/wiki/Computer_Science" class="mw-redirect" title="Computer Science">computer science</a> and <a href="/wiki/Artificial_Intelligence" class="mw-redirect" title="Artificial Intelligence">artificial intelligence</a>.</td>
</tr>
<tr>
<td>1968</td>
<td>Noriko Umeda</td>
<td>His team at Japan's Electrotechnical Laboratory developed the first <a href="/wiki/Speech_synthesis" title="Speech synthesis">text-to-speech synthesis</a> system.<sup id="cite_ref-33" class="reference"><a href="#cite_note-33">[33]</a></sup></td>
</tr>
<tr>
<td>1950~</td>
<td><a href="/wiki/An_Wang" title="An Wang">An Wang</a></td>
<td>Made key contributions to the development of <a href="/wiki/Magnetic_core" title="Magnetic core">magnetic core</a> memory.</td>
</tr>
<tr>
<td>1955<br />
1960s<br />
1974</td>
<td><a href="/wiki/Willis_Ware" title="Willis Ware">Willis Ware</a></td>
<td>Co-designer of <a href="/wiki/JOHNNIAC" title="JOHNNIAC">JOHNNIAC</a>. Chaired committee that developed the <a href="/wiki/FTC_Fair_Information_Practice" title="FTC Fair Information Practice">Code of Fair Information Practice</a> and led to the <a href="/wiki/Privacy_Act_of_1974" title="Privacy Act of 1974">Privacy Act of 1974</a>. Vice-chair of the Privacy Protection Study Commission.</td>
</tr>
<tr>
<td>1968</td>
<td><a href="/wiki/Adriaan_van_Wijngaarden" title="Adriaan van Wijngaarden">Adriaan van Wijngaarden</a></td>
<td>Developer of the <a href="/wiki/W-grammar" class="mw-redirect" title="W-grammar">W-grammar</a> first used in the definition of <a href="/wiki/ALGOL_68" title="ALGOL 68">ALGOL 68</a></td>
</tr>
<tr>
<td>1949</td>
<td><a href="/wiki/Maurice_Wilkes" title="Maurice Wilkes">Maurice Wilkes</a></td>
<td>Built the first practical <a href="/wiki/Stored_program" class="mw-redirect" title="Stored program">stored program</a> computer (<a href="/wiki/EDSAC" class="mw-redirect" title="EDSAC">EDSAC</a>) to be completed and for being credited with the ideas of several high-level programming language constructs.</td>
</tr>
<tr>
<td>1981<br />
1985</td>
<td><a href="/wiki/Sophie_Wilson" title="Sophie Wilson">Sophie Wilson</a></td>
<td>Wrote <a href="/wiki/BBC_Basic" class="mw-redirect" title="BBC Basic">BBC Basic</a> programming language. Also designed the <a href="/wiki/ARM_architecture" title="ARM architecture">ARM architecture</a> and Firepath processors</td>
</tr>
<tr>
<td>1970<br />
1978</td>
<td><a href="/wiki/Niklaus_Wirth" title="Niklaus Wirth">Niklaus Wirth</a></td>
<td>Designed the <a href="/wiki/Pascal_programming_language" class="mw-redirect" title="Pascal programming language">Pascal</a>, <a href="/wiki/Modula-2" title="Modula-2">Modula-2</a> and <a href="/wiki/Oberon_(programming_language)" title="Oberon (programming language)">Oberon</a> programming languages.</td>
</tr>
<tr>
<td>1938<br />
1945</td>
<td><a href="/wiki/Konrad_Zuse" title="Konrad Zuse">Konrad Zuse</a></td>
<td>Built the first digital freely programmable computer, the <a href="/wiki/Z1_(computer)" title="Z1 (computer)">Z1</a>. Built the first functional program-controlled computer, the <a href="/wiki/Z3_(computer)" title="Z3 (computer)">Z3</a>.<sup id="cite_ref-34" class="reference"><a href="#cite_note-34">[34]</a></sup> The Z3 was proven to be Turing-complete in 1998. Produced the world's first commercial computer, the <a href="/wiki/Z4_(computer)" title="Z4 (computer)">Z4</a>. Designed the first high-level programming language, <a href="/wiki/Plankalk%C3%BCl" title="Plankalkül">Plankalkül</a>.</td>
</tr>
