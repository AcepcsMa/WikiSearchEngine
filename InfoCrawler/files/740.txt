			<h1 id="firstHeading" class="firstHeading" lang="en">List of mayors of Nuremberg</h1>
									<div id="bodyContent" class="mw-body-content">
									<div id="siteSub" class="noprint">From Wikipedia, the free encyclopedia</div>
								<div id="contentSub"></div>
												<div id="jump-to-nav" class="mw-jump">
					Jump to:					<a href="#mw-head">navigation</a>, 					<a href="#p-search">search</a>
				</div>
				<div id="mw-content-text" lang="en" dir="ltr" class="mw-content-ltr"><div class="mw-parser-output"><p>This is a <b>list of <a href="/wiki/Mayor" title="Mayor">mayors</a> of <a href="/wiki/Nuremberg" title="Nuremberg">Nuremberg</a></b> (<i>Oberbürgermeister der Stadt Nürnberg</i>) since 1818:</p>
<ul>
<li>1818–1821: <a href="/wiki/Christian_Gottfried_Lorsch" title="Christian Gottfried Lorsch">Christian Gottfried Lorsch</a></li>
<li>1821–1853: <a href="/w/index.php?title=Jakob_Friedrich_Binder&amp;action=edit&amp;redlink=1" class="new" title="Jakob Friedrich Binder (page does not exist)">Jakob Friedrich Binder</a></li>
<li>1854–1867: <a href="/w/index.php?title=Maximilian_von_Waechter&amp;action=edit&amp;redlink=1" class="new" title="Maximilian von Waechter (page does not exist)">Maximilian von Waechter</a></li>
<li>1867–1891: <a href="/w/index.php?title=Otto_Freiherr_von_Stromer&amp;action=edit&amp;redlink=1" class="new" title="Otto Freiherr von Stromer (page does not exist)">Otto Freiherr von Stromer</a></li>
<li>1892–1913: <a href="/w/index.php?title=Georg_Ritter_von_Schuh&amp;action=edit&amp;redlink=1" class="new" title="Georg Ritter von Schuh (page does not exist)">Georg Ritter von Schuh</a> (<a href="/wiki/Free-minded_People%27s_Party_(Germany)" title="Free-minded People's Party (Germany)">FVP</a>)</li>
<li>1913–1919: <a href="/wiki/Otto_Ge%C3%9Fler" class="mw-redirect" title="Otto Geßler">Otto Geßler</a> (<a href="/wiki/Progressive_People%27s_Party_(Germany)" title="Progressive People's Party (Germany)">FVP</a>)</li>
<li>1920–1933: <a href="/w/index.php?title=Hermann_Luppe&amp;action=edit&amp;redlink=1" class="new" title="Hermann Luppe (page does not exist)">Hermann Luppe</a> (<a href="/wiki/German_Democratic_Party" title="German Democratic Party">DDP</a>)</li>
<li>1933–1945: <a href="/w/index.php?title=Willy_Liebel&amp;action=edit&amp;redlink=1" class="new" title="Willy Liebel (page does not exist)">Willy Liebel</a> (<a href="/wiki/Nazi_Party" title="Nazi Party">NSDAP</a>)</li>
<li>1945: <a href="/w/index.php?title=Julius_R%C3%BChm&amp;action=edit&amp;redlink=1" class="new" title="Julius Rühm (page does not exist)">Julius Rühm</a> (NSDAP)</li>
<li>1945: <a href="/w/index.php?title=Martin_Treu&amp;action=edit&amp;redlink=1" class="new" title="Martin Treu (page does not exist)">Martin Treu</a> (<a href="/wiki/Social_Democratic_Party_of_Germany" title="Social Democratic Party of Germany">SPD</a>)</li>
<li>1945–1948: <a href="/w/index.php?title=Hans_Ziegler_(politician)&amp;action=edit&amp;redlink=1" class="new" title="Hans Ziegler (politician) (page does not exist)">Hans Ziegler</a> (SPD)</li>
<li>1948–1951: <a href="/w/index.php?title=Otto_Ziebill&amp;action=edit&amp;redlink=1" class="new" title="Otto Ziebill (page does not exist)">Otto Ziebill</a> (SPD)</li>
<li>1952–1957: <a href="/w/index.php?title=Otto_B%C3%A4rnreuther&amp;action=edit&amp;redlink=1" class="new" title="Otto Bärnreuther (page does not exist)">Otto Bärnreuther</a> (SPD)</li>
<li>1957–1987: <a href="/w/index.php?title=Andreas_Urschlechter&amp;action=edit&amp;redlink=1" class="new" title="Andreas Urschlechter (page does not exist)">Andreas Urschlechter</a> (SPD)</li>
<li>1987–1996: <a href="/w/index.php?title=Peter_Sch%C3%B6nlein&amp;action=edit&amp;redlink=1" class="new" title="Peter Schönlein (page does not exist)">Peter Schönlein</a> (SPD)</li>
<li>1996–2002: <a href="/wiki/Ludwig_Scholz" title="Ludwig Scholz">Ludwig Scholz</a> (<a href="/wiki/Christian_Social_Union_of_Bavaria" class="mw-redirect" title="Christian Social Union of Bavaria">CSU</a>)</li>
<li>since 2002: <a href="/wiki/Ulrich_Maly" title="Ulrich Maly">Ulrich Maly</a> (SPD)</li>
</ul>
