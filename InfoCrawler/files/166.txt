			<h1 id="firstHeading" class="firstHeading" lang="en">Xafecopy Trojan</h1>

<p><b>Xafecopy Trojan</b> is a <a href="/wiki/Malware" title="Malware">malware</a> <a href="/wiki/Software" title="Software">software</a> of Ubsod family targeting the <a href="/wiki/Android_operating_system" class="mw-redirect" title="Android operating system">Android operating system</a>, identified first in September 2017 by <a href="/wiki/Cybersecurity" class="mw-redirect" title="Cybersecurity">cybersecurity</a> and antivirus provider <a href="/wiki/Kaspersky_Lab" title="Kaspersky Lab">Kaspersky</a> targeting significantly on India based android devices. According to reports, Xafecopy infected at least 4,800 users in just a month in 47 countries with over 37.5 percent of damage identified by Kaspersky Lab products targeting India, followed by attacks on devices in Russia, Turkey, and Mexico.<sup id="cite_ref-times_1-0" class="reference"><a href="#cite_note-times-1">[1]</a></sup><sup id="cite_ref-2" class="reference"><a href="#cite_note-2">[2]</a></sup><sup id="cite_ref-3" class="reference"><a href="#cite_note-3">[3]</a></sup></p>
<p></p>

<h2><span class="mw-headline" id="History">History</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Xafecopy_Trojan&amp;action=edit&amp;section=1" title="Edit section: History">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Xafecopy was first discovered by Kaspersky in 2017, when it was found to have infected thousands of android based devices in India. The malware embedded in useful apps like BatteryMaster with a backdoor to download <a href="/wiki/Malicious_code" class="mw-redirect" title="Malicious code">malicious code</a> covertly onto the device.<sup id="cite_ref-4" class="reference"><a href="#cite_note-4">[4]</a></sup> Once the malicious codes are activated it starts to click on web pages with <a href="/wiki/Wireless_Application_Protocol" title="Wireless Application Protocol">Wireless Application Protocol</a> (WAP) billing method, following which the malware subscribes the phone to a number of services which charges money.<sup id="cite_ref-times_1-1" class="reference"><a href="#cite_note-times-1">[1]</a></sup><sup id="cite_ref-5" class="reference"><a href="#cite_note-5">[5]</a></sup></p>
<p>Xafecopy has been found using <a href="/wiki/Javascript" class="mw-redirect" title="Javascript">Javascript</a> file names which was previously used by infamous Ztorg Trojan, triggering speculation of a possibility of code sharing between cyber criminal gangs.<sup id="cite_ref-6" class="reference"><a href="#cite_note-6">[6]</a></sup><sup id="cite_ref-7" class="reference"><a href="#cite_note-7">[7]</a></sup></p>
<h2><span class="mw-headline" id="Operation">Operation</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Xafecopy_Trojan&amp;action=edit&amp;section=2" title="Edit section: Operation">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>Xafecopy disguises itself as an useful app, often a battery optimizer.<sup id="cite_ref-8" class="reference"><a href="#cite_note-8">[8]</a></sup> It operates by clicking on web pages with WAP billing system, which is a form of mobile payment system, charged directly to the mobile bill. WAP billing is particularly associated with downloading mobile entertainment content. The malware works in WAP-enabled android devices over a <a href="/wiki/GPRS" class="mw-redirect" title="GPRS">GPRS</a> or <a href="/wiki/3G" title="3G">3G</a> wireless connection. The malware is based on the Ubsod family, and detected by <a href="/wiki/Kaspersky_Lab" title="Kaspersky Lab">Kaspersky Lab</a> as Trojan-Clicker-AndroidOS.Xafekopy. The malware receives the WAP billing URL addresses of the web pages through a command-and-control server. Once the URL address is received at the device, the malware clicks on the WAP billing links, which initiates a WAP session with the server, which then obtains the user's <a href="/wiki/MSISDN" title="MSISDN">MSISDN</a> and charges directly to the user's mobile carrier bill and subscribes to unwanted paid services.<sup id="cite_ref-kaspersky_9-0" class="reference"><a href="#cite_note-kaspersky-9">[9]</a></sup><sup id="cite_ref-times_1-2" class="reference"><a href="#cite_note-times-1">[1]</a></sup><sup id="cite_ref-10" class="reference"><a href="#cite_note-10">[10]</a></sup></p>
<p>The malware is claimed to be using technology which bypasses <a href="/wiki/Captcha" class="mw-redirect" title="Captcha">captcha</a> systems.<sup id="cite_ref-times_1-3" class="reference"><a href="#cite_note-times-1">[1]</a></sup> According to Kaspersky Lab, the malware is sharing significant amount of codes obtained from other significant malware.<sup id="cite_ref-11" class="reference"><a href="#cite_note-11">[11]</a></sup></p>
<p>Modified versions of Xafecopy were also identified, where the malware is capable of sending <a href="/wiki/SMS" title="SMS">SMS</a> from the device to premium-rate phone numbers and to delete incoming SMS from the mobile network provider hiding alerts about balance deduction by reading incoming messages and checking for words like "subscription".<sup id="cite_ref-kaspersky_9-1" class="reference"><a href="#cite_note-kaspersky-9">[9]</a></sup></p>
<p>The malware is also capable of switching an user from <a href="/wiki/WiFi" class="mw-redirect" title="WiFi">WiFi</a> connection to mobile data, as <a href="/wiki/WAP_billing" title="WAP billing">WAP billing</a> works only when the user is connected to a mobile connection.<sup id="cite_ref-kaspersky_9-2" class="reference"><a href="#cite_note-kaspersky-9">[9]</a></sup></p>
