<p><b>DBLP</b> is a <a href="/wiki/Computer_science" title="Computer science">computer science</a> <a href="/wiki/Bibliography" title="Bibliography">bibliography</a> website. Starting in 1993 at the <a href="/wiki/University_of_Trier" title="University of Trier">University of Trier</a>, <a href="/wiki/Germany" title="Germany">Germany</a>, it grew from a small collection of HTML files<sup id="cite_ref-Ley2009_1-0" class="reference"><a href="#cite_note-Ley2009-1">[1]</a></sup> and became an organization hosting a <a href="/wiki/Database" title="Database">database</a> and <a href="/wiki/Logic_programming" title="Logic programming">logic programming</a> bibliography site. DBLP listed more than 3.66 million journal articles, conference papers, and other publications on computer science in July 2016, up from about 14,000 in 1995.<sup id="cite_ref-2" class="reference"><a href="#cite_note-2">[2]</a></sup> All important journals on computer science are tracked. Proceedings papers of many conferences are also tracked. It is mirrored at three sites across the <a href="/wiki/Internet" title="Internet">Internet</a>.<sup id="cite_ref-dblp_3-0" class="reference"><a href="#cite_note-dblp-3">[3]</a></sup><sup id="cite_ref-4" class="reference"><a href="#cite_note-4">[4]</a></sup><sup id="cite_ref-5" class="reference"><a href="#cite_note-5">[5]</a></sup></p>
<p>For his work on maintaining DBLP, <a href="/wiki/Michael_Ley" title="Michael Ley">Michael Ley</a> received an award from the <a href="/wiki/Association_for_Computing_Machinery" title="Association for Computing Machinery">Association for Computing Machinery</a> and the VLDB Endowment Special Recognition Award in 1997.</p>
<p><i>DBLP</i> originally stood for <i>DataBase systems and Logic Programming</i>. As a <a href="/wiki/Backronym" title="Backronym">backronym</a>, it has been taken to stand for <i>Digital Bibliography &amp; Library Project</i>;<sup id="cite_ref-6" class="reference"><a href="#cite_note-6">[6]</a></sup> however, it is now preferred that the acronym be simply a name, hence the new title "The DBLP Computer Science Bibliography".<sup id="cite_ref-dblp-faq_7-0" class="reference"><a href="#cite_note-dblp-faq-7">[7]</a></sup></p>
<p>Users of dblp remain unaffected by some additional attributes in the DTD as of February 2016, which are meant to support future versions of the data file. Thus consumers of the raw dblp.xml file should update their local dblp.dtd file, according to the notice on the home page, as of February 2016.<sup id="cite_ref-8" class="reference"><a href="#cite_note-8">[8]</a></sup></p>

<p><i>DBL-Browser</i> (<i>Digital Bibliographic Library Browser</i>) is a utility for browsing the DBLP website. The browser was written by Alexander Weber in 2005 at the <a href="/wiki/University_of_Trier" title="University of Trier">University of Trier</a>. It was designed for use off-line in reading the DBLP, which consisted of 696,000 bibliographic entries in 2005 (and in 2015 has more than 2.9 million).</p>
<p>DBL-Browser is <a href="/wiki/GPL" class="mw-redirect" title="GPL">GPL</a> <a href="/wiki/Software" title="Software">software</a>, available for download from <a href="/wiki/SourceForge" title="SourceForge">SourceForge</a>. It uses the <a href="/wiki/XML" title="XML">XML</a> <a href="/wiki/Document_Type_Definition" class="mw-redirect" title="Document Type Definition">DTD</a>. Written in <a href="/wiki/Java_programming_language" class="mw-redirect" title="Java programming language">Java programming language</a>, this code shows the bibliographic entry in several types of screens, ranging from graphics to text:</p>
<ul>
<li>Author page</li>
<li>Article page</li>
<li>Table of contents</li>
<li>Related conferences / journals</li>
<li>Related authors (graphic representation of relationships)</li>
<li>Trend analysis (graphics histogram)</li>
</ul>
<p>DBLP is similar to the bibliographic portion of <a href="/wiki/Arxiv.org" class="mw-redirect" title="Arxiv.org">arxiv.org</a> which also links to articles. DBL-Browser provides a means to view some of the associated computer science articles.</p>
