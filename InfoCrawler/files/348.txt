<p><b>Raabs an der Thaya</b> is a municipality with 3,114 inhabitants in <a href="/wiki/Waidhofen_an_der_Thaya_(district)" class="mw-redirect" title="Waidhofen an der Thaya (district)">Waidhofen an der Thaya (district)</a> in the <a href="/wiki/Waldviertel" title="Waldviertel">Waldviertel</a> of <a href="/wiki/Lower_Austria" title="Lower Austria">Lower Austria</a>, near the <a href="/wiki/Austria" title="Austria">Austrian</a> border with the <a href="/wiki/Czech_Republic" title="Czech Republic">Czech Republic</a>. About 27.6 percent of the municipality is forested.</p>
<p>The <a href="/wiki/German_Thaya" title="German Thaya">German Thaya</a> and the <a href="/wiki/Moravian_Thaya" title="Moravian Thaya">Moravian Thaya</a> unite to form the <a href="/wiki/Thaya" title="Thaya">Thaya</a> at Raabs. <a href="/wiki/Burg_Raabs_an_der_Thaya" title="Burg Raabs an der Thaya">Raabs castle</a>, built in the last half of the 11th century, has been exonymically called <i>Rakous</i> (formerly <i>Rakús</i>) by neighbouring <a href="/wiki/Czechs" title="Czechs">Czechs</a>, which is the origin of the <a href="/wiki/Czech_language" title="Czech language">Czech</a> and <a href="/wiki/Slovak_language" title="Slovak language">Slovak</a> name for Austria as a whole, <i>Rakousko</i>/<i>Rakúsko</i>.</p>
